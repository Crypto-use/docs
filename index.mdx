Nesta documentação você terá tudo o que precisa para integrar com a API da CryptoUse. Não se preocupe, a CryptoUse é feita de devs pra devs, então saiba que será muito fácil integrar.

## O que é a CryptoUse?

A CryptoUse é um gateway de pagamento de criptomoedas que oferece uma forma simples e eficiente de integrar pagamentos em blockchain em seu aplicativo ou site. Nossa plataforma facilita depósitos, saques e pagamentos com diferentes criptomoedas e redes.

Nossa plataforma auxilia você a processar pagamentos em cripto de forma simples e descomplicada. Pense que pegamos toda a parte difícil de uma integração blockchain e abstraímos pra você em uma API bem tranquila de integrar.

## API descomplicada

**Nossa API** é **baseada em intenção**. Cada operação em cada endpoint é previsível pois cada URL representa exatamente o que você lê.

**Nossa API** é **indempotente**, o que significa que você pode fazer a mesma requisição várias vezes sem efeitos colaterais indesejados. Isso garante que suas operações sejam seguras e previsíveis.

Recebemos JSON, retornamos JSON sempre com o mesmo formato. O que auxilia a integração independente da linguagem utilizada.

## Recursos principais

### Pagamentos Onchain
Aceite pagamentos em criptomoedas diretamente nas redes blockchain como Ethereum, BSC e Polygon.

### Pagamentos Pix
Gere QR codes Pix para pagamentos em reais, perfeito para usuários que não possuem criptomoedas.

### Lightning Network
Aceite pagamentos em Bitcoin via Lightning Network para transações rápidas e de baixo custo.

### Conversão Crypto → Pix
Converta criptomoedas para Pix automaticamente, facilitando pagamentos em estabelecimentos que aceitam apenas Pix.
